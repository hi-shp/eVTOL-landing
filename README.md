# MASS-eVTOL Landing Simulator

## **MediaPipe와 PD제어를 활용한 해상 특수목적 비행체(eVTOL) 자율착륙 시뮬레이션 구현**  
본 프로젝트는 해상 불확실성(Wave & Ship Pitch) 환경에서 자율운항 특수목적 비행체(eVTOL/UAV)가 안전하게 선박 갑판에 착함할 수 있도록 지원하는 실시간 원격 제어(HMI) 및 자율 착륙 시뮬레이터입니다.

---

## 🚀 주요 기능 (Key Features)

* **실시간 손가락 스켈레톤 추적 (MediaPipe HMI):** 웹캠을 통해 사용자 손목과 중지의 벡터를 추출하여 비행체의 Pitch(기울기)를 직관적으로 조종합니다.
* **불규칙 파랑 및 선체 거동 모사:** 다중 사인파 중첩 알고리즘을 활용하여 실제 해상 환경의 롤링/피칭 운동을 실시간으로 구현했습니다.
* **FCS 자율 착륙 시스템 (Auto-Landing):** `T` 키 전환 시, 선박의 동적 위치와 기울기 오차를 0으로 수렴시키는 피드백 제어 알고리즘이 구동됩니다.
* **물리 엔진 기반 충돌/안착 판정:** 갑판 방정식과의 경계면 실시간 연산을 통해 착함 순간의 수직 하강 속도(Velocity) 및 각도 편차(Attitude Error)를 계측하여 Safe Touchdown 여부를 판정합니다.
* **전술 비행 관제 대시보드 (HUD):** 실시간 텔레메트리 데이터, 피치 편차 오차 그래프, 수직 하강 속도 그래프를 겹치지 않게 배치하여 시각적 직관성을 높였습니다.
